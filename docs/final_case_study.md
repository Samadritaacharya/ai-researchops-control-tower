# Final Case Study

## Proactive AI Coding Assistant Research Program

This independent portfolio case study models the management of an AI/ML software-engineering research program for an IDE-like developer tool.

## Research Tracks

1. Intent detection
2. Suggestion quality
3. Privacy and data assessment
4. Product handover readiness

## Management Focus

The project demonstrates uncertainty scoring, risk tracking, task completeness analysis, stakeholder coordination, experiment tracking, and research-to-product handover.
